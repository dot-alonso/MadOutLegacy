# MadOutLegacy

BepInEx плагин для Steam-версии игры **MadOut2 BigCityOnline**, полностью восстанавливающий онлайн и офлайн функционал

### Подробная информация о возможностях мода, запуске и настройке сервера находится в разделе [Wiki](https://github.com/dot-alonso/MadOutLegacy/wiki)
---

## Возможности

- Офлайн режим игры - полностью убрана зависимость от сервера
- Убрано обязательное требование Steam
- Запуск своего игрового сервера с гибкой настройкой
- Поддержка всех онлайн-режимов (FreeRoam, RP, Гонки, Копы против Бандитов)
- Возможность создания своих трасс и локаций для онлайн-гонок и копов против бандитов
- Монеты и алмазы можно бесплано получить в магазине ("+" в главном меню)
- Дополнительные горячие клавиши (см. Wiki)
- Снятие лимита FPS

## Совместимость

Полностью поддерживается **только последняя Steam версия игры** `9.4`

Для более ранних версий (`4.9 - 9.2`) доступен режим совместимости (Compatibility Mode), обеспечивающий только офлайн функционал. На данный момент режим совместимости не гарантирует полную и стабильную работу ранних версий игры.

## Установка

* Скачайте посделнюю **оригинальную** версию игры из Steam (или здесь)
* Установите BepInEx 5 в папку с игрой и запустите игру 1 раз
* Скачайте zip файл из раздела Releases и распакуйте в корневую папку игры

## Сборка

**Требования:**

- MadOut2 9.4 с установленным BepInEx 5
- .NET SDK / MSBuild

**Процесс сборки:**

1. Установите BepInEx 5 в папку с игрой и запустите игру 1 раз.
2. Соберите проект, указав путь к корневой папке игры:

```bat
dotnet build -c Release -p:GameDir="path\to\game\dir" -p:CopyToPlugins=true
```
