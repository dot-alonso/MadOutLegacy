# Описание

**MadOutLegacy** - проект, целью которого является полное восстановление онлайн и офлайн функционала Steam-версии игры MadOut2 BigCityOnline для обеспечения оригинального игрового опыта и сохранения истории игры.

Проект представляет собой модификацию для игры и распространяется в виде плагина для BepInEx 5

# Функции и возможности (подробно):

* **Мод убирает обязательную привязку игры к онлайн-серверам** и обеспечивает полную работу в офлайн режиме.
* **Мод делает игру Steam-less**, то есть убирает требование работающего Steam для запуска игры. При этом базовый Steam-функционал остается рабочим, например Steam может отслеживать ваш онлайн-статус и подсчитывать время, проведенное в игре.
* **Мод позволяет запустить игру в режиме сервера** и **легко подключиться к нему через отдельное меню**. Сервер имеет несколько режимов и гибкую настройку (см. подробно в разделе Сервер)
* **Сервер поддерживает все основные игровые режимы** (FreeRoam, RP, Гонки и "Копы против Бандитов").
  *Онлайн-паркур в планах на восстановление в будущих релизах и пока поддерживается лишь косвенно.*
* **Сервер имеет полноценный headless-режим**, не требует графики для запуска и способен работать в Windows или Linux консоли. Мод исправляет нерабочий сервер в nographics режиме на Linux версии игры.
* Возможность запуска Mirror сервера для отдачи клиенту списка серверов и их состояния (см. в разделе Сервер).
* **Возможно создание собственных трасс для онлайн-гонок и зон для режима "Копы против Бандитов".**
  Оригинальные так называемые ивенты (т.е. трассы и зоны) полностью утеряны, т.к загружались с сервера и не содержатся в клиенте игры, поэтому загрузка собственных ивентов является обязательным для работы режимов. В текущем релизе представлено лишь несколько ивентов как шаблон для создания своих (см. раздел Создание ивентов)
  **Позднее будет добавлено много готовых ивентов** для обеспечения готового игрового опыта. Вы можете предлагать и публиковать свои ивенты в разделе Discussions.
* Мод имеет небольшой инструмент (Debug Overlay) для удобного создания своих ивентов (копирования и создания цепочки координат камеры). Инструмент пока представлен в базовом виде и планируется улучшаться в будущем.
* Монеты и алмазы можно получить бесплатно по простому нажатию на офферы в магазине (кнопка "+" в правом верхнем углу главного меню)
* Мод добавляет несколько горячих клавиш:
  - `F5` - быстрое переключение погоды
  - `F6` - мгновенный спавн последнего выбранного авто (без тайи-аутов)
  - `F12` - экстренное отключение от сервера (в случаях, если игра зависла на черном экране)
* Мод позволят снять ограничение FPS (ограничение ставится на 10000)

# Параметры запуска

Мод имеет множество параметров (аргументов запуска), передаваемых запускному файлу игры. Они используются в основном для запуска сервера, но есть параметры и для клиента.

**Аргументы передаются в виде:**

```
game.exe -arg1 -arg2 -arg3
```

#### Для клиента:


| Аргумент                | Назначение                                                                                                          | Принимаемые значения       |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| -debugoverlay                   | Запустить игру с Debug Overlayдля создания ивентов                                            | -                                             |
| -connectIP:[значение]   | Указать адрес игрового сервера для прямого (Direct) подключения               | IP/домен игрового сервера |
| -connectPort:[значение] | Указать порт игрового сервера для прямого (Direct) подключения                 | Порт игрового сервера      |
| -useMaster                      | Автоматически использовать указанный/сохраненный mirror сервера           | -                                             |
| -masterIP:[значение]    | Указать адрес mirror сервера                                                                               | IP/домен mirror сервера           |
| -masterPort:[значение]  | Указать порт mirror сервера                                                                                 | Порт mirror сервера                |
| -forceFullMode                  | Принудительно запустить мод в полном режиме (для версий игры ниже 9.4) | -                                             |
| -forceCompatMode                | Принудительно запустить мод в режиме совместимости (Compatibilty Mode)           | -                                             |

#### Для сервера:


| Аргумент            | Назначение                                                                                                                                                                                                                                                      | Принимаемые значения                          |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| -bend_GameServer            | Запустить игру в режиме сервера                                                                                                                                                                                                                | -                                                                |
| -batchmode -nographics      | Использовать headless-режим (рекомендуется)                                                                                                                                                                                                 | -                                                                |

#### Примеры использования:

Запустить игру с debug overlay:

```
game.exe -debugoverlay
```

Запустить игру, заранее указав игровой сервер:

```
game.exe -connectIP:1.2.3.4 -connectPort:7800
```

Запустить игру, заранее указав mirror сервер:

```
game.exe -useMaster -masterIP:1.2.3.4 -masterPort:35000
```

Запустить **сервер** в режиме base (только game server):

```
game.exe -bend_GameServer -batchmode -nographics -port:7800 -maxConn:32 -Mode:Race
```

Запустить **сервер** в режиме master (game server + mirror):

```
game.exe -bend_GameServer -batchmode -nographics -port:7800 -maxConn:32 -Mode:Race -masterServer
```

Запустить **сервер** в режиме master, указав собственные файлы с конфигами:

```
game.exe -bend_GameServer -batchmode -nographics -port:7800 -maxConn:32 -Mode:Race -masterServer -serversList:myservers.json -eventsList:myevents.json
```

Запустить **сервер** в графическом режиме (не рекомендуется):

```
game.exe -bend_GameServer -port:7800 -maxConn:32 -Mode:Race
```

[!info] В комплекте с модом лежат скрипты `runServer.cmd` и `runServer.sh` для быстрого запуска сервера. Они имеют удобную настройку всех параметров внутри себя.

# Режим совместимости

[!info] Мод имеет полную поддержку только версии игры `9.4`, которая является последней для Steam.

При запуске на более ранних версиях игры мод запускается в режиме совместимости (Compatibility Mode) и пытается обеспечить их базовую работу (без онлайн-функционала).

[!warning] Стабильная работа игры и её офлайн функционала в режиме совместмости **не гарантируется**!

Режим совместимости и поддержку более старых версий игры планируется улучшать в будущем.

# Подключение к серверу

Подключиться к серверу можно через окно **Server Connection**

Имеется вкладка **Mirror** и **Direct**

При подключении к Mirror игра получает с него список серверов и отображает в меню. Адрес и порт сохраняются. Можно включить автоматическое подключение к Mirror сразу при запуске игры, поставив галочку на "Auto resolve on start game"

В Direct же можно напрямую подключиться к игровому серверу по IP и порту. При этом статус онлайна в меню отображаться не будет.
